PurFood Nutrition Plan

This is the project root index.html. During development with Create React App, the app is served using public/index.html and bootstrapped by src/index.js.

To run the app in development, use:
npm install then npm start

When the dev server is running, open:
http://localhost:3000

Files used by the React app:

  • public/index.html – HTML template with <div id="root">
  • src/index.js – React entry point
  • src/App.js – App shell loading components from src/components/
